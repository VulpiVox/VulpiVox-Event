# Fusing Narrative Semantics for Financial Volatility Forecasting

**Bank of America | Quant Speaker Series Seminar**

**Speaker:** Yaxuan Kong, University of Oxford, Department of Engineering Science

**Date:** March 18, 2026

## Abstract

Multi-modal deep learning framework (M2VN) for financial volatility forecasting that fuses unstructured news narratives with structured price data while maintaining temporal integrity.
